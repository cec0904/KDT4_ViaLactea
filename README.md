# KDT4_ViaLactea

Unreal Engine C++ source-code backup for the KDT4 team project.

This repository is intended for portfolio and source-code review purposes only.

## Included
- Source
- Config
- Project plugins
- `.uproject`

## Excluded
- Content
- Binaries
- Intermediate
- Saved
- DerivedDataCache

The Content folder is intentionally excluded due to file size and asset licensing.
Therefore, this repository may not run as a complete Unreal Engine project by itself.
